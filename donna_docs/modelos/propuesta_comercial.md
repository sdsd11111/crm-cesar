# Modelo de Propuesta Comercial Estratégica

Donna, usa esta estructura para generar propuestas comerciales formales y estratégicas. Este es el modelo oficial del CRM.

## Instrucciones de Redacción
- **Tono:** Directo, profesional, orientado a resultados.
- **Enfoque:** Mostrar el problema (el negocio no aparece en Google) y la solución (más páginas = más oportunidades).
- **Regla:** Prohibido prometer posiciones garantizadas. Prohibido lenguaje genérico aburrido.

## Estructura del Documento

### 1. Portada
```markdown
# PROPUESTA COMERCIAL
Sitio Web + Posicionamiento en Google para {{NEGOCIO}}

Dirigido a: {{CLIENTE}}
Fecha: {{FECHA}}
```

### 2. El Problema (Gancho inicial)
**Título:** "Tus clientes ya están buscándote en Google… pero no te encuentran"
**Contenido:** Crear 4-6 ejemplos de búsquedas reales que la gente hace en {{CIUDAD}} relacionadas con {{ESPECIALIDADES}}. "¿Y quién aparece? No {{NEGOCIO}}."

### 3. Fortalezas del Cliente
**Título:** "Tienes todo para ganar… pero Google no lo sabe"
**Contenido:** Listar 4-6 fortalezas del negocio (reputación, ubicación, especialidades). "Pero si no apareces en la primera página de Google, simplemente no existes."

### 4. La Solución (Concepto Clave)
**Título:** "La solución: un sitio web que abre múltiples puertas de entrada en Google"
**Concepto:** Cada página funciona como una vía independiente. Más páginas = más palabras clave = más clientes.

### 5. Los Planes
Donna, consulta el `catalogo_productos.md` para los detalles de los planes:
- **Plan Emprendedor ($250)**
- **Plan Crecimiento ($500)**
- **Plan Pro ($700) [RECOMENDADO]**
- **Plan Posicionamiento ($1,000)**

### 6. Tabla Comparativa
Generar una tabla que compare: Precio, Páginas, Impacto Google, Blog, Dominio/Hosting, Reservas.

### 7. Próximos Pasos (Fijo)
1. Confirmar el plan.
2. Firma de contrato (se activa con el 60% inicial).
3. Reunión estratégica.
4. Entrega de material.
5. Desarrollo y Lanzamiento.

### 8. Cierre Premium
Reforzar la oportunidad. {{CIUDAD}} está lleno de personas buscando {{ESPECIALIDADES}}. Solo falta que Google los muestre antes que a los demás.
